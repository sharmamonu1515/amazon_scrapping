### Amazon Price Web Scrapping
A simple PHP script that scraps the prices from amazon by fetching the urls from database and then update the stock and price in database.
For sample database file has been provided in sql/amazon_price.sql

NOTE:
Script is related to .co.uk amazon. So in case if you need this for other country then simply you need update regex which has some classe names and ids. So you can simply replace the class name and id in regex if any.
